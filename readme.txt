to open the project:
run the server.py file and it will generate a link to the server that hosts our website.

the website home page shows some general information about the hospital.
you can sign up as a doctor but you can not use an email as a doctor unless the admin accepts

your request which is sent automatically to him once you sign up as a doctor.

the admin email is inserted manually into the database before runnig the project.

once the admin accepts your request you can sign in with your email and add appointments that
patients can book.

any one can sign up as a patient and book appointments.

every user has a profile that includes his information and appointments.

you can also send your feedback and messages to doctors.
